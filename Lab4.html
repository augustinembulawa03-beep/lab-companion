Overview

This lab has three parts. First, you will plot the output characteristics of a 2N7000 MOSFET: how drain current ID changes with VDS at fixed gate voltages.

Second, you will plot the transfer characteristics: how ID changes with VGS at fixed VDS = 10 V. From these graphs you will extract VT, gm, and ro, then compare with datasheet values. An optional task uses the MOSFET as a switch to blink an LED with a square-wave gate signal.

Equipment Checklist

Safety & Common Mistakes

  • ⚠ This lab uses two channels of the Keithley 2231A simultaneously: Channel 1 controls VGS via VGG, Channel 2 controls VDS via VDD. Do not swap these connections.
  • ⚠ The oscilloscope is used for DC readings here. Press Measure and select Mean. Set voltage scale slightly above expected values to avoid clipping.
  • ⚠ For drain current, DMM must be in series between RD and drain terminal (positive lead to resistor side, negative lead to drain).
  • ⚠ Use white 3A + black port for current mode (SHIFT then DCV), and top-right red + black for voltage mode.

Theory Mini-Lesson

2–4 minute read – key ideas only

In this lab you generate two key graphs. The output characteristics show ID vs VDS at fixed VGS (1.1 V, 1.3 V, 1.5 V). For small VDS, current rises approximately linearly (triode region). At larger VDS, current flattens toward saturation.

The transfer characteristics show ID vs VGS at fixed VDS = 10 V. Below threshold voltage VT, little current flows; above VT, ID rises rapidly.

Definitions used in this lab:
VT: gate-source threshold voltage where drain current begins to rise.
gm = ΔID/ΔVGS (at constant VDS).
ro = ΔVDS/ΔID (at constant VGS).

Unlike a BJT, a MOSFET gate is insulated by an oxide layer, so gate current IG is effectively zero. This is why an input characteristic IG vs VGS is not normally plotted.

Step-by-Step Procedure

  1. Task 1 — Output characteristics (family of curves)
    Build the Figure 4 style circuit with 2N7000. Use Channel 1 (VGG) for gate voltage and Channel 2 (VDD) for drain supply. Connect scope CH1 to VGS, CH2 to VDS; read DC values using Measure > Mean. Insert DMM in series between RD and drain to measure ID. Set VGS = 1.1 V, then sweep VDS from 1 V to 10 V in 1 V steps, recording ID. Repeat full sweep for VGS = 1.3 V and 1.5 V.
  2. Task 2 — Transfer characteristics
    Keep the same circuit. Fix VDS at 10 V. Sweep VGS from 0 to 1.55 V (0.1 V steps, including 1.55 V), recording ID for each point. Plot VGS vs ID and extract VT from the graph. Compute gm from the linear-slope region and ro from the saturation region of output curves.
    Lab tip: If the drain current does not increase when increasing gate voltage, check that the MOSFET source pin is connected to ground. This is a frequent wiring issue.
    Expected Result: MOSFET threshold voltage is typically between 1 V and 3 V depending on device type.
    No drain current below threshold is normal.
    If current flows below threshold, check wiring orientation.
  3. Task 3 (optional) — MOSFET LED switching demonstration
    Build the optional switching circuit (Figure 5 style). Set signal generator to 3.3 V square wave at 1 Hz. Use LED L-53YD with 560 Ω resistor in drain path and observe blinking (cut-off vs conduction switching states).
