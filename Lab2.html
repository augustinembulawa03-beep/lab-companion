Overview

In this lab you will measure the current–voltage (I–V) relationship of a silicon diode in forward and reverse bias, and plot the characteristic that you observe.

This matters because real diodes do not behave like ideal on/off switches. Their knee voltage, leakage current and power dissipation limits all affect how power supplies, protection circuits and digital logic behave in real hardware.

Safety & Common Mistakes

  • ⚠ Before switching on, set the power-supply current limit to 100 mA.
  • ⚠ For voltage measurement, set DMM to DCV and use the top-right red and right black ports (parallel measurement).
  • ⚠ For current measurement, place DMM in series using SHIFT + DCV, with white 3A and right black ports.
  • ⚠ In reverse-bias testing, swap the PSU polarity at the supply side only; do not rewire the whole breadboard.

Theory Mini-Lesson

2–4 minute read – key ideas only

A diode ideally conducts in one direction only. In practice, very little current flows until a "knee" voltage is reached (about 0.6–0.7 V for silicon, about 0.2–0.3 V for germanium). Beyond this point the current rises quickly for small changes in voltage.

In reverse bias, only a tiny leakage current flows until breakdown, which is far above the voltages used in this lab. A series resistor is therefore used to limit the current so the diode stays within its safe operating area.

In this experiment you compare two devices: silicon (1N4148) and germanium (1N34A). Silicon typically turns on around 0.6–0.7 V, while germanium turns on around 0.2–0.3 V. The different knee behaviour is linked to different semiconductor material properties.

The power in the diode at any instant is:

P = VD × ID

Step-by-Step Procedure

  1. Task 1: Complete diode datasheet values before building
    Fill in the key datasheet values for both 1N4148 and 1N34A: rated wattage, maximum forward current, and peak reverse voltage.
    Hint: prepare this table first so you can compare your measured values against limits later.
  2. Task 2: Forward-bias measurements for silicon diode (1N4148)
    Build PSU to 100 Ω resistor to diode. Set current limit to 100 mA before enabling output. Set target current values (100, 50, 20, 10, 5, 2, 1 mA) and record diode voltage at each point. Then set fixed voltages (0.5 V, 0.25 V, 0.1 V) to capture low-current data.
    Hint: voltage measurement uses DMM in parallel (DCV, top-right red + right black). Current measurement uses series insertion (SHIFT + DCV, white 3A + right black).
  3. Task 3: Reverse-bias sweep and repeat for germanium (1N34A)
    For reverse bias, swap PSU + and − leads on the supply side and measure from 0 V to −10 V in −2 V steps. Repeat forward + reverse process for 1N34A, then plot both I–V curves in Excel with clear axis labels and titles.
    Hint: keep the same measurement method and table format for both diodes so comparison is valid.
