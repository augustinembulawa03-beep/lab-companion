Overview

This lab has two tasks. In Task 1 you will build a CMOS inverter using a VP2106/TP2104 PMOS and a BS170 NMOS, then use oscilloscope XY mode to capture the transfer characteristic curve (VOUT vs VIN).

From that curve you will estimate VOH, VOL, VIL, and VIH and calculate noise margins. In Task 2 you will measure propagation delay, then add 220 pF and 0.1 µF output capacitive loads to observe how delay increases in real digital circuits.

Safety & Common Mistakes

  • ⚠ Identify PMOS and NMOS pin configurations from datasheets before wiring; swapped drain/source can break operation and damage components.
  • ⚠ In XY mode, keep both channels DC-coupled and scale slightly above 5 V; incorrect scale can clip or compress the transfer curve.
  • ⚠ In delay measurements, adjust time base so input/output edge delay is visible; too coarse a time base hides tpHL and tpLH differences.

Theory Mini-Lesson

2–4 minute read – key ideas only

In a CMOS inverter, PMOS pulls output high when VIN is low, while NMOS pulls output low when VIN is high. The transfer characteristic (VOUT vs VIN) shows this transition.

Noise-margin terms used in this lab:
VOH = highest output interpreted as logic 1, VOL = lowest output interpreted as logic 0.
VIL = maximum input still interpreted as logic 0, VIH = minimum input interpreted as logic 1.
NML = VIL − VOL, NMH = VOH − VIH.

Propagation delay: measured between 50% input transition and 50% output transition. You measure tpHL (input high causes output low) and tpLH (input low causes output high).

Capacitive loading effect: output capacitance must be charged/discharged each transition. Larger C means longer switching time, so delay increases (demonstrated with 220 pF and 0.1 µF).

XY mode reminder: CH1 drives horizontal axis (VIN), CH2 drives vertical axis (VOUT), so you observe direct transfer characteristics rather than time-domain traces.

Step-by-Step Procedure

  1. Task 1A — Datasheet setup before building
    Look up VP2106/TP2104 and BS170 datasheets. Record pin configuration and fill Table 1: gate threshold voltage (min/max), turn-on delay, and turn-off delay for both transistors.
  2. Task 1B — Transfer characteristic in XY mode
    Build the inverter (PMOS source to VDD, NMOS source to ground, gates tied to input, drains tied to output). Set function generator to triangular wave: low = 0 V, high = 5 V, frequency = 60 Hz. Connect scope CH1 to input and CH2 to output, both DC-coupled. Switch scope to XY mode and adjust scales to view full 0–5 V range. Capture curve and estimate VIL, VIH. Then calculate NML and NMH.
    Common mistake: Students often forget to switch the oscilloscope into XY mode before capturing the transfer curve. This results in a flat or incorrect display.
    Expected Result: Output should remain near VDD for low input voltages and near 0 V for high input voltages.
    Transition region should occur near mid-supply voltage.
    Unexpected behaviour usually indicates wiring or supply connection error.
  3. Task 2 — Propagation delay and capacitive loading
    Switch scope back to time-base mode. Set input to 5 Vpp square wave at 1 kHz. Measure tpHL and tpLH without output capacitor. Add 220 pF from output to ground and repeat. Replace with 0.1 µF and repeat again. Capture a scope screenshot with visible delay.

Post-Lab Reflection

  • What was the effect of capacitive loading on propagation delay times? Compare no-cap, 220 pF, and 0.1 uF results and explain the trend.
  • Where did your transfer characteristic switch most sharply, and what does that steep region imply about noise rejection?
  • Why is VOH slightly below VDD and VOL slightly above 0 V in real measurements?
  • Were tpHL and tpLH equal? If not, suggest why PMOS and NMOS switching paths may give different delays.
