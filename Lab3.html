Safety & Common Mistakes

  • ⚠ This lab uses two separate Keithley 2231A channels: one for VBB (base) and one for VCC (collector). Do not tie both channels incorrectly to the same terminal.
  • ⚠ As VBB changes, adjust VCC to keep VCE constant (2 V then 3 V).
  • ⚠ When switching DMM between voltage and current mode, move the probe lead to the correct port (white 3A for current, top-right red for voltage).
  • ⚠ Confirm transistor pin numbering before inserting into breadboard to avoid E/B/C reversal.

Theory Mini-Lesson

2–4 minute read – key ideas only

1) Double-diode analogy for BJT identification

A BJT contains two PN junctions, so it behaves like two diodes sharing the base terminal. For an NPN transistor, both base-emitter and base-collector junctions conduct when the red DMM probe is on base and black probe on emitter/collector (about 0.6 V for silicon). Reversing probes gives OL.

If forward readings appear when red is on the shared terminal, the transistor is NPN. If forward readings appear when black is on the shared terminal, it is PNP.

2) Common-emitter input characteristics

In this lab, you measure the input curve by plotting IB vs VBE at fixed VCE. Common-emitter is widely used because it can provide voltage, current, and power gain.

VCC sets collector supply, VCE is collector-emitter voltage, and VBE is base-emitter voltage that controls base current.

3) Dynamic input resistance

The input characteristic looks diode-like. In the steeper region, dynamic input resistance can be estimated from ΔVBE / ΔIB, i.e., the reciprocal of the local IB-VBE slope.

Step-by-Step Procedure

  1. Task 1 — Double-diode identification for six transistors
    Hold each transistor with flat/printed side facing you: left pin = 1, middle = 2, right = 3. Set DMM to diode function and record all pin pairs: 1 to 2, 2 to 1, 2 to 3, 3 to 2, 3 to 1, 1 to 3 (red probe on first number). Identify base terminal and classify NPN/PNP. Repeat for 2N2222A, ZTX689B, ZTX751, BC327, BC337, TIP31C. Then look up KSP2222 datasheet and fill VCBO, VEBO, and minimum hFE at VCE = 10 V, IC = 1 mA.
  2. Task 2 — Build common-emitter input-characteristic circuit (BC337-40)
    Build Figure 7 style circuit with two PSU channels: Channel 1 = VBB (base path through R1 = 100 kΩ), Channel 2 = VCC (collector path). Use DMM 1 in series with R1 to measure IB (white 3A + black port, SHIFT then DCV), and DMM 2 in parallel for VBE/VCE (top-right red + black port, DCV).
    Common mistake: Students sometimes swap collector and emitter connections on the BC337 transistor. This allows the circuit to partially operate but produces incorrect measurements.
    Expected Result: Base-Emitter voltage should normally be between 0.6 V and 0.7 V for a silicon transistor.
    Values significantly lower suggest insufficient base current.
    Values significantly higher indicate wiring error.
  3. Task 3 — Sweep VBB and keep VCE constant
    First set VCE = 2 V and keep it constant while sweeping VBB (0.1 V to 5 V per table). Record IB and VBE. Then repeat entire sweep at VCE = 3 V. Plot IB vs VBE curves (both VCE values on one graph), and calculate dynamic input resistance using ΔVBE/ΔIB from the linear region.
